# InstantAIGate.Runtimes.win-x64.Cpu

Native CPU runtimes for win-x64. This package provides the unmanaged binaries required to run **InstantAIGate** middleware on Windows utilizing CPU inference.

InstantAIGate is a lightweight middleware providing a self-hosted, monitored foundation for local AI applications. 

## 📦 Requirements

* **Framework**: .NET 10.
* **OS**: Windows Server 2022 / Windows 10 & 11 (64-bit).
* **Architecture**: x86-64 with AVX2 instruction sets.
* **Engine**: Built upon a highly optimized, stripped-down custom compilation of `llama.cpp`.
