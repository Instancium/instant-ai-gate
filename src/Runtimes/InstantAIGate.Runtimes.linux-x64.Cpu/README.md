# InstantAIGate.Runtimes.linux-x64.Cpu

Native CPU runtimes for linux-x64. This package provides the unmanaged binaries required to run **InstantAIGate** middleware on Linux utilizing CPU inference.

InstantAIGate is a lightweight middleware providing a self-hosted, monitored foundation for local AI applications. 

## 📦 Requirements

* **Framework**: .NET 10.
* **OS**: Ubuntu 22.04 LTS (or any glibc-compatible modern Linux distribution).
* **Architecture**: x86-64 with AVX2 instruction sets.
* **Engine**: Built upon a highly optimized, stripped-down custom compilation of `llama.cpp`.
